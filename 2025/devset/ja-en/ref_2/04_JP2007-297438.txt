1. A semiconductive urethane elastomer-forming composition comprising a main agent (A) containing an isocyanate group-terminated urethane prepolymer obtained by reacting a polyether polyol (a1) with an organic polyisocyanate (a2), and a curing agent (B) containing a polyol (b1), a conductive agent (b2), and an antistatic agent (b3),
wherein the conductive agent (b2) is a conductive carbon, the antistatic agent (b3) is an ionic salt composed of a quaternary ammonium cation having an aliphatic unsaturated hydrocarbon group having 10 to 20 carbon atoms and one hydroxyalkyl group, and a bis(trifluoromethanesulfonyl)imide anion, and the respective contents of the conductive agent (b2) and the antistatic agent (b3) relative to the total amount of the main agent (A) and the curing agent (B) are (b2): 0.1 to 1% by mass, and (b3): 0.001 to 10% by mass.

2. A semiconductive urethane elastomer-forming composition comprising a main agent (A) containing an isocyanate group-terminated urethane prepolymer obtained by reacting a polyether polyol (a1) with an organic polyisocyanate (a2), and a curing agent (B) containing a polyol (b1), a conductive agent (b2), and an antistatic agent (b3),
wherein the conductive agent (b2) is a conductive carbon, the antistatic agent (b3) is an ionic salt composed of a quaternary ammonium cation having an aliphatic unsaturated hydrocarbon group having 10 to 20 carbon atoms and two hydroxyalkyl groups, and a bis(trifluoromethanesulfonyl)imide anion, and the respective contents of the conductive agent (b2) and the antistatic agent (b3) relative to the total amount of the main agent (A) and the curing agent (B) are (b2): 0.1 to 1% by mass, and (b3): 0.001 to 10% by mass.
