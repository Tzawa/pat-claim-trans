1. A system, comprising:
memory including machine-readable instructions; and
one or more processors configured, in response to executing the machine-readable instructions, to perform operations comprising:
identifying, from among a plurality of user profiles accessible to the one or more processors and associated with a corresponding plurality of users, a first user profile of first a user located within a cabin of a vehicle;
detecting a first hand movement gesture performed by the first user while the first user is located within the cabin;
in response to detecting the first hand movement gesture, identifying, by accessing the first user profile, a first gesture preference specified by the first user and included in the first user profile, the first gesture preference indicating a first user-customized control command associated with the first hand movement gesture; and
generating a first control signal based on the first user-customized control command, the first control signal causing a first modification of a function or setting of a vehicle subsystem of the vehicle according to the first user-customized control command, wherein generation of a second control signal causing a second modification of a function or setting of the vehicle subsystem that differs from the first modification occurs in response to detecting performance of a second hand movement gesture by a second user when the second user is located within the cabin, the second hand movement gesture matching the first hand movement gesture, the second control signal determined by a second user-customized control command associated with the second hand movement gesture and indicated by a second gesture preference included in a second user profile of the second user, the second gesture preference differing from the first gesture preference.

2. The system of claim 1, wherein the operations performed by the one or more processors further comprise:
in response to receiving an override associated with the first gesture preference, determining a standard control command associated with the first hand movement gesture, the standard control command differing from the first user-customized control command; and
generating a third control signal based on the standard control command, the third control signal causing a third modification of the function or setting of the vehicle subsystem according to the standard control command, the third modification differing from the first modification.
